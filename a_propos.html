À propos de Jobeo

Le Projet

Dans le cadre de notre formation d'ingénieur au CESI, nous avons développé Web4all, une plateforme web dédiée à la mise en relation entre étudiants et entreprises. Ce projet a pour objectif de centraliser les offres de stages et d'alternances pour faciliter les démarches de nos camarades et des recruteurs partenaires.

Objectifs Pédagogiques

Ce site est le résultat d'un travail collaboratif visant à mettre en pratique nos compétences techniques :

  • Architecture Web : Structuration propre et responsive.
  • Gestion de Version : Travail d'équipe via Git et GitHub (Workflow par branches).
  • Expérience Utilisateur : Création d'une interface fluide pour la consultation des offres.

L'Équipe Jobeo

Nous sommes un groupe de 5 étudiants passionnés par le développement web. Ce projet nous permet d'expérimenter les réalités d'un cycle de développement, de la gestion du backlog à la mise en production des fonctionnalités, comme le module actuel de gestion des entreprises.
